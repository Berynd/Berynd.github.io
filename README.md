# Portfolio — Gabriel Berinde

Portfolio personnel déployé via GitHub Pages.

## Déploiement sur GitHub Pages

1. Crée un dépôt GitHub nommé **`Berynd.github.io`** (tu en as déjà un !)
2. Pousse les fichiers :

```bash
git init
git add .
git commit -m "Initial portfolio"
git remote add origin https://github.com/Berynd/Berynd.github.io.git
git push -u origin main
```

3. Active GitHub Pages dans les **Settings** → **Pages** → source : `main` branch `/root`

Le site sera disponible sur : **https://berynd.github.io**
